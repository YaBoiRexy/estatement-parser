[estatement-parser](https://yaboirexy.github.io/estatement-parser/)

Expects 1 file: Estatement entries, e.g. 2026-01-01,"STEVE'S POKE",16.01,,1234********1234

Uses keywords.csv and colors.csv, which:
1. Map maps key words to category, e.g. STEVE'S POKE, Food
2. Map category to color, e.g. Food, #34a853
respectively.

Visit link above, upload file, and click execute.
