[estatement-parser](https://yaboirexy.github.io/estatement-parser/)

Expects 3 files: 

1. Maps maps text to category, e.g. STEVE'S POKE, Food
2. Maps category to color, e.g. Food, #34a853
3. Estatement entries, e.g. 2026-01-01,"STEVE'S POKE",16.01,,1234********1234

Visit link above, upload files, and click execute.
